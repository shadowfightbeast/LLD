## 🧠 Problem-Solving Practice

# Classic Low-Level Design Problems

Challenge yourself with these classic low-level design problems:

1. 🅿️ **Parking Lot**  
   Design a system to manage parking spaces, including allocation, tracking, and payment functionalities.

2. ♟ **2-Player Games (e.g., Chess)**  
   Create a system to handle game state, player moves, and game rules for two-player games.

3. 🏢 **Elevator System**  
   Build a system to manage elevator requests, scheduling, and prioritization for multiple floors and elevators.

4. 🗂 **Caching Library**  
   Design a caching mechanism to efficiently store and retrieve frequently accessed data, optimizing performance.

5. 🎟 **Ticket Booking System**  
   Develop a system to handle ticket reservations, availability, and payment for events or transportation.

6. 🂡 **Blackjack Game**  
   Create a system to simulate a blackjack game, including deck management, player actions, and game rules.

7. 📋 **Booking Management System**  
   Design a system to manage various bookings, including scheduling, cancellations, and notifications.

8. 🏧 **ATM**  
   Build an ATM system for managing user accounts, transactions, and security features for cash withdrawals and deposits.

9. 🥤 **Vending Machine Design**  
   Create a system to handle inventory, transactions, and user interactions for vending machines.

10. 🚗 **Car Rental System**  
    Design a system for managing car rentals, including vehicle availability, reservations, and customer management.

11. 🏏 **CricBuzz Design**  
    Build a system for live cricket scores, match updates, and player statistics.

12. 💻 **StackOverflow Design**  
    Develop a Q&A platform for users to post questions, provide answers, and manage user interactions.

13. 🚕 **Uber/Ola Design**  
    Design a ride-sharing system to manage driver and rider matching, trip tracking, and payment processing.

14. 📦 **Warehouse Management System (WMS)**  
    Create a system to track inventory, manage warehouse operations, and optimize storage and retrieval processes.

15. 🛒 **E-Commerce System**  
    Design a system for online shopping, including product listings, cart management, and order processing.

16. 🛒 **Quick-Commerce System (e.g., Zepto, Blinkit)**  
    Build a system for ultra-fast delivery services, managing inventory, orders, and delivery logistics.

17. 🛵 **Food Delivery System (e.g., Zomato/Swiggy)**  
    Create a system to handle food orders, restaurant listings, delivery management, and customer interactions.

18. 🧾 **Billing and Subscription System**  
    Design a system for managing recurring payments, billing cycles, and subscription management.

19. 🏦 **Banking System**  
    Create a system for managing user accounts, transactions, loans, and ATMs.

20. 📦 **Package Delivery System (e.g., FedEx)**  
    Build a system to manage package tracking, delivery scheduling, and logistics.

21. 📈 **Stock Trading System**  
    Design a system for trading stocks, managing orders, and tracking market data.

22. 📱 **Messaging/Chat Application (e.g., WhatsApp)**  
    Build a messaging system to handle real-time communication, user profiles, and chat history.

23. 🧾 **Billing and Subscription System**  
    Create a system for managing recurring payments, billing cycles, subscriptions, and payment failures.

24. 💼 **Job Portal System (e.g., LinkedIn)**  
    Build a system where users can post and apply for jobs, create profiles, and receive job recommendations.

25. 🖥 **Content Management System (CMS) (e.g., WordPress)**  
    Build a system like WordPress that allows users to create, edit, and manage website content dynamically.

26. 📧 **Email System (e.g., Gmail)**  
    Design a system for sending and receiving emails, managing inboxes, and applying filters.

27. 🛍 **Digital Wallet (e.g., PayPal)**  
    Build a system to manage user accounts, handle payments, and support peer-to-peer money transfers.

28. 🔧 **Bug Tracking System (e.g., Jira)**  
    Design a system for tracking project issues, managing workflows, and assigning tasks to developers.

29. 🚴‍♂️ **Fitness Tracking Application (e.g., Strava)**  
    Build a system to track workouts, display user progress, and provide social sharing features.

## 🔨 Creational Design Patterns

These patterns deal with object creation, helping to create objects in a way suitable to the situation.

1. 🏭 **Factory Method**
2. 🏭 **Abstract Factory Pattern**
3. 🛠 **Builder Pattern**
4. 🧬 **Prototype Pattern**
5. 🔁 **Singleton Pattern**

## 🏗 Structural Design Patterns

These patterns deal with object composition and structure to ensure efficient object relationships.

1. 🔌 **Adapter Pattern**
2. 🌉 **Bridge Pattern**
3. 🔍 **Filter Pattern**
4. 🧩 **Composite Pattern**
5. 🎨 **Decorator Pattern**
6. 🏢 **Facade Pattern**
7. 🪶 **Flyweight Pattern**
8. 🕵️‍♂️ **Proxy Pattern**

2. 📜 **Template Method Pattern**
3. 🔗 **Chain of Responsibility**
4. 📝 **Command Pattern**
5. 🔄 **Iterator Pattern**
6. 🧠 **Strategy Pattern**
