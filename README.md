## 🔨 Creational Design Patterns

These patterns deal with object creation, helping to create objects in a way suitable to the situation.

1. 🏭 **Factory Method**
2. 🏭 **Abstract Factory Pattern**
3. 🛠 **Builder Pattern**
4. 🧬 **Prototype Pattern**
5. 🔁 **Singleton Pattern**

---

## 🏗 Structural Design Patterns

These patterns deal with object composition and structure to ensure efficient object relationships.

1. 🔌 **Adapter Pattern**
2. 🌉 **Bridge Pattern**
3. 🔍 **Filter Pattern**
4. 🧩 **Composite Pattern**
5. 🎨 **Decorator Pattern**
6. 🏢 **Facade Pattern**
7. 🪶 **Flyweight Pattern**
8. 🕵️‍♂️ **Proxy Pattern**

## 🧩 Behavioral Design Patterns

These patterns focus on communication between objects, making your software adaptable and flexible.

1. 🗣 **Interpreter Pattern**
2. 📜 **Template Method Pattern**
3. 🔗 **Chain of Responsibility**
4. 📝 **Command Pattern**
5. 🔄 **Iterator Pattern**
6. 🧠 **Strategy Pattern**
7. 🛂 **Visitor Pattern**
